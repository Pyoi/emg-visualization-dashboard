# emg-visualization-dashboard
2020年版お試し筋電のを保存しました
ビッタリーノみたいなやつで使ってみたものです。

A simple Streamlit dashboard for visualizing EMG time-series data.

## Features
Moving average smoothing
- Activity-based color coding
- Peak detection
- Clean and simple UI

## How to Run
pip install -r requirements.txt  
streamlit run app.py
